Artemis-2009 - Version 2.6

(English follows)

Nouveautés

  1. La nouvelle version comprend un nouveau module de prélèvement basé sur les placettes de suivi du MRNF (Feature #1363)
  2. Une bar de progression apparaît lors de l'exportation des données (Feature #1113)
  3. Les options de changements climatiques ont été corrigées (Bug #1045)
  4. L'interpolation 5 ans lors de l'exportation a été corrigée (Bug #1047)
  5. Le générateur de nombre aléatoire qui permet de simuler le nombre de recrues en mode stochastique a été corrigé (Bug #1284)
  6. Le module de prévision du dhp des recrues a été corrigé (Bug #933)
  7. Un modulateur de traitement permet de moduler le prélèvement par un facteur variant entre -80% et +160% (Feature #1439)
  8. Il est maintenant possible de sélectionner plus d'un format d'exportation (Feature #1332)
  9. L'option "Peuplement ou strate résiduelle" a été modifiée afin de mieux réfléter les conditions de croissance après coupe (Bug #1491)
  10. Par défaut, les graphiques affichent maintenant les résultats par hectare (Feature #994)
  11. Un module de billonnage SYBILLE permet maintenant de réaliser un billonnage virtuel des tiges de huit espèces (Feature #2333)
  12. Un nouveau format d'exportation permet de sauvegarder les résultats du billonnage virtuel (Feature #2335)
  13. Il est maintenant possible d'exporter en format .csv (Feature #2334)

New features

  1. This version includes a new harvest module based on the MRNF monitoring plots (Feature #1363)
  2. A progress bar shows up when exporting the data (Feature #1113)
  3. The climate change options have been corrected (Bug #1045)
  4. The 5-yr interpolation option in the export tool has been corrected for inconsistent values (Bug #1047)
  5. The random number generator for the number of recruits in stochastic mode has been corrected (Bug #1284)
  6. The recruit dbh module has been corrected for inconsistent values (Bug #933)
  7. A treatment modulator enables a harvest modulation between -80% and +160% (Feature #1439)
  8. Several export formats can be selected at the same time (Feature #1332)
  9. The "Residual stand or stratum" option has been modified in order to better predict the growth conditions following partial cutting (Bug #1491)
  10. By default, the graphs now show the results per hectare (Feature #994)
  11. A bucking module SYBILLE allows to perform a virtual bucking for eight species (Feature #2333)
  12. A new export format allows to save the results of the virtual bucking (Feature #2335)
  13. It is now possible to export to .csv files (Feature #2334)
